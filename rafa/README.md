## 🎯 Concepto / Concept

**Español:** Un videojuego de plataformas estilo Mario con un personaje plomero italiano. En el juego aparecen dragones, unicornios, y dinosaurios verdes pequeños (como Yoshi).

**English:** A Mario-style platformer video game with an Italian plumber character. The game features dragons, unicorns, and small green dinosaurs (like Yoshi).
